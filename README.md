# Ilir Krasniqi – Portfolio

Personal portfolio site for **Ilir Krasniqi**, Software Engineer from Pristina, Kosovo. Showcases experience, key projects, tech stack, education, and contact.

## Description

A single-page portfolio built with vanilla HTML, CSS, and JavaScript. Features a resizable sidebar navigation, dark/light theme (with preference saved in `localStorage`), and accessible, responsive layout. Sections include:

- **Home** – Intro and highlights  
- **Experience** – Work history  
- **Key Projects** – Markdown Editor, Label Designer, Planet Accounting, Lead Generator, and others  
- **Tech Stack** – React, TypeScript, Node.js, full-stack development  
- **Education** – Background and learning  
- **Contact** – How to get in touch  

No build step required. Open `index.html` in a browser or serve the folder with any static host (e.g. Netlify, GitHub Pages).

## Run locally

```bash
# Option 1: open directly
start index.html

# Option 2: serve with a simple HTTP server (e.g. Python)
python -m http.server 8000
# Then visit http://localhost:8000
```

## Tech

- HTML5, CSS3, JavaScript (vanilla)  
- Material Symbols (icons)  
- No frameworks or bundlers
